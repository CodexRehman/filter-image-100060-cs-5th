PAGE 4: ABSTRACT
=========================================
"Image Pro" is a cutting-edge Android application designed to provide users with professional-level image enhancement tools directly on their mobile devices. Developed using Kotlin and Jetpack Compose, the application integrates advanced features such as real-time camera filtering, sophisticated color matrix transformations, and custom text overlays.

The app leverages the CameraX library for a seamless camera experience and utilizes hardware-accelerated RenderEffects (on supported Android versions) to apply complex filters without performance lag. This project demonstrates the implementation of modern Android architectural components and UI design principles to create a functional and aesthetically pleasing utility tool.

PAGE 6: INTRODUCTION
=========================================
In the modern digital age, mobile photography is more prevalent than ever. Users demand high-quality editing tools that are both powerful and easy to use. "Image Pro" is developed to fill the gap between basic camera apps and complex editing software.

The application provides a "What You See Is What You Get" (WYSIWYG) experience, where users can see exactly how a filter will look on their photo before they even take it. By focusing on performance and user experience, Image Pro offers a set of curated filters that cater to various artistic needs, from classic Noir and Sepia to modern HD and Vivid enhancements.

PAGE 7: PROBLEM STATEMENT & MOTIVATION
=========================================
Traditional image editing often requires multiple steps: taking a photo, opening an editor, applying changes, and saving. Many existing apps are cluttered with ads or require subscriptions for basic features.

Motivation:
The motivation behind "Image Pro" was to create a clean, ad-free, and high-performance tool that utilizes the latest Android capabilities. I wanted to explore the boundaries of real-time image processing on mobile hardware and provide a tool that is useful for students and hobbyists alike.

PAGE 9-11: TECHNOLOGY STACK
=========================================
4.1 KOTLIN:
Kotlin is a cross-platform, statically typed, general-purpose programming language with type inference. It is designed to interoperate fully with Java. Kotlin is the preferred language for Android development due to its conciseness and safety features.

4.2 JETPACK COMPOSE:
Jetpack Compose is Android’s modern toolkit for building native UI. It simplifies and accelerates UI development on Android. It uses a declarative approach, meaning you describe your UI by calling a series of functions that transform data into a UI hierarchy.

4.3 CAMERAX:
CameraX is a Jetpack library, built to help make camera app development easier. It provides a consistent and easy-to-use API surface that works across most Android devices, with backward compatibility to Android 5.0 (API level 21).

4.4 MEDIASTORE:
The MediaStore API is used to interact with the device's media database. In this project, it is used to save processed images into the public "Pictures" directory, ensuring they appear in the system gallery immediately.
